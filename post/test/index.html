简要教程

combo-tree是一款jQuery带多选和过滤功能的树状结构下拉框插件。通过该插件，可以在下拉框中生成指定数据结构的目录树，提供单选和多选，以及过滤功能。它的特点有：

  • 在下拉框中显示树状结构。
  • 支持单选和多选。
  • 返回选择数据的 title 或 id 数组。
  • 带过滤功能。
  • 数据源使用json格式数据。
  • 可以通过键盘控制。

使用方法

在HTML文件中引入下面的文件。

<link rel="stylesheet" href="css/style.css">
<script src="js/jquery.min.js" type="text/javascript"></script>
<script src="js/icontains.js"type="text/javascript"></script>
<script src="js/comboTreePlugin.js"type="text/javascript"></script>

<input type="text" id="example" placeholder="Select">

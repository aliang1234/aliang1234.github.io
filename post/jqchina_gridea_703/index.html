简要教程

这是一款基于owl-carousel的响应式文章卡片设计效果。该效果以卡片的方式展示最新文章，带文章缩略图和描述信息等，鼠标滑过缩略图时带放大效果。并且整个文章卡片以旋转木马的方式来回滚动。

owl-carousel是一款强大的响应式jQuery旋转木马插件。它支持触摸设备，支持IE9+的所有现代浏览器。

使用方法

在页面中引入owl.carousel.min.css和owl.carousel.min.js文件。

<link rel="stylesheet" href="css/owl.carousel.min.css"/>
<script type="text/javascript" src="js/owl.carousel.min.js"></script>
HTML结构

一张文章卡片的完整HTML结构如下：

<div class="post-slide">
<div class="post-img">
<a href="#"><img src="img/1.jpg" alt=""></a>
</div>
<div class="post-content">
<h3 class="post-title"><a href="#">最新文章</a></h3>
<p class="post-description">
......
</p>
<ul class="post-bar">
<li><i class="fa fa-calendar"></i> 2016-7-28</li>
<li>
<i class="fa fa-folder"></i>
<a href="#">标签1</a>
<a href="#">标签2</a>
<a href="#">标签3</a>
</li>
</ul>
<a href="#" class="read-more">点击查看</a>
</div>
</div>

JavaScript

在页面DOM元素加载完毕之后，通过下面的方法来初始化owl-carousel插件。

$(document).ready(function() {
$("#news-slider").owlCarousel({
items:3,
itemsDesktop:[1199,2],
itemsDesktopSmall:[980,2],
itemsMobile:[600,1],
pagination:false,
navigationText:false,
autoPlay:true
});
});

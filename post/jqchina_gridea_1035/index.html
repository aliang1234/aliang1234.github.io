HTML结构

该滑块的HTML结构非常简单，实际上就是一个3D立方体。a、b、c、d分别为立方体的左右和上前4个面，div.slider-range-min是滑块的凹槽。实际的滑块是一个<input>元素。

<section id="content">
<h1>......</h1>
<div class="cube">
<div class="a"></div>
<div class="b"></div>
<div class="c"></div>
<div class="d"></div>
<div id="slider-range-min"></div>
</div>
<input type="text" id="amount" />
</section>

Javascript

在jQuery代码中使用jQueryUI的slider()方法来制作滑块。并使数值和滑块进行联动。

$(function () {
$('#slider-range-min').slider({
range: 'min',
value: 50,
min: 0,
max: 100,
slide: function (event, ui) {
$('#amount').val('$' + ui.value + ',000');
$('.a, .b, .c, .d').width(ui.value + '\%');
}
});
$('.ui-slider-handle').text('<>');
$('#amount').val('$' + $('#slider-range-min').slider('value') + ',000');
});
